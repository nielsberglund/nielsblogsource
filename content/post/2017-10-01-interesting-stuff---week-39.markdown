* [Identifying SqlConnection objects in a dump][1]. [Remus Rusanu][2] is doing cool WinDbg debugging to finding out  ADO.Net connection pool exhaust issues.
* [What made Porting Microsoft SQL Server to Linux to be the right move after all][4]. Slava Oks writes about Microsoft's "secret sauce", which made porting SQL Server to Linux possible.

## Data Science

* [News Roundup from Microsoft Ignite][6]. Microsoft's Ignite conference took place this week, and the [Revolution Analytics][re] guys posts some data science highlights.
